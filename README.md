# USDRUB Telegram Bot

Telegram bot that provides latest US Dollar to Russian Ruble exchange rate. 

## Features

* Forex rate
* Moscow Exchange rate ([MOEX](https://www.moex.com/en/))
* Russian Central Bank exchange rate
* Cash exchange rates
